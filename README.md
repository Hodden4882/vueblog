# Vueblog
作品名稱:vueblog

是一個功能很簡單的blog,實現前後分離,主要使用Springboot+vue製作

也是我第一個作品,1-2週後也許有第二個作品.


# 前端
*核心框架Vue :2.6.11

*Element UI

*前後端異步請求 :axios

*其他: 第三方markdown組件

# 後端
*核心框架: SpringBoot :2.6.1

*安全框架: Shiro (有用到redis

*Token: jwt

*dao層框架 :MyBatisPlus

JDK: 11

# 項目啟動
1.使用sql文件,創建數據庫

2.啟動redis

3.修改後端配置文件,並運行後端

4.使用npm運行前端
